Getting Started

The Timetable Generator allows you to create professional-looking timetables for schools, work schedules, or personal planning. Here's a quick overview of how to get started:

  1. Configure General Settings: Set your name, initials, tutor group, and choose which days and periods to display
  2. Add Events: Create lessons or appointments by clicking "Add New Event"
  3. Generate Timetable: Click "Generate Timetable" to see your schedule
  4. Download: Save your timetable as a PNG image

üí° Quick Tip

Start with one of our example templates by downloading them from the home page, then modify them to suit your needs!

General Settings

The General Settings panel lets you customize how your timetable appears:

Short Name

Your full name as you want it to appear on the timetable

Initials

Your initials for a more compact display option

Tutor Group

Your class or group identifier (e.g., "10A", "Team Alpha")

Name Display

Choose how your name appears: None, Short Name, Initials, Tutor Group, or Both

Days and Periods

Select which days of the week and which periods to include in your timetable. This allows you to create timetables for different schedules:

  • School Schedule: Monday-Friday with periods 1-6
  • Work Schedule: Monday-Friday with custom time slots
  • Weekend Schedule: Saturday-Sunday for personal activities

Adding Events & Lessons

Events are the individual lessons, meetings, or activities that make up your timetable.

Creating a New Event

  1. Click the "Add New Event" button
  2. Select the Day and Period from the dropdown menus
  3. Enter the Subject (e.g., "Mathematics", "Team Meeting")
  4. Add the Room location (e.g., "Room 101", "Conference Room A")
  5. Enter the Staff name (teacher, instructor, or meeting organizer)
  6. Choose a Color to help categorize your events
  7. Click "Save Event"

Managing Events

Once you've added events, you can:

  • Edit: Click the "Edit" button next to any event to modify its details
  • Delete: Click the "Delete" button to remove an event
  • Duplicate: Use the "Duplicate" button to copy an event to different times

üé® Color Coding Tip

Use consistent colors for different subjects or types of activities. For example: Blue for Math, Green for Science, Red for English.

Using Presets

Presets allow you to save commonly used event configurations and quickly apply them to new time slots.

Creating a Preset

  1. Add an event with all the details you want to save
  2. In the events list, click "Save as Preset" on that event
  3. Give your preset a memorable name
  4. The preset will appear in the Lesson Presets panel

Using a Preset

  1. When adding a new event, select a preset from the dropdown at the top of the modal
  2. All fields will be automatically filled with the preset values
  3. You can still modify any field before saving

üìö Preset Ideas

Create presets for: Regular subjects with the same teacher and room, Recurring meetings, Break times, Study periods, or Laboratory sessions.

Import & Export

Save your work and share timetables using the JSON import/export feature.

Exporting Your Timetable

  1. Click "Manage JSON" in the Current Lessons panel
  2. Click "Export to Textarea" to see your data
  3. Click "Download JSON File" to save it to your computer

Importing a Timetable

  1. Click "Manage JSON"
  2. Either paste JSON data into the textarea and click "Import JSON"
  3. Or use "Choose JSON File" to upload a saved file

Troubleshooting

Common Issues

Timetable Not Generating

Solution: Make sure you have at least one day and one period selected, and at least one event added.

Events Not Showing

Solution: Check that the event's day and period are selected in the General Settings panel.

JSON Import Failed

Solution: Ensure the JSON data is valid and was exported from this application. Check for any syntax errors.

Download Not Working

Solution: Make sure you've generated the timetable first, and check your browser's download permissions.
